# Poketmon_Dictionary(포켓몬도감)

![](snapshot/Frame%201.png)

## 사용한 라이브러리

- flutter_riverpod, 상태관리 라이브러리
- riverpod_annotation, 상태관리 라이브러리(지금까지는 안쓴듯)
- dio, 통신 라이브러리
- html, html parser를 위한 라이브러리
- lottie, image loading 시 사용하기 위한 애니메이션 라이브러리
- go_router, router용
